# Clipboard Image Saver

Windowsのクリップボードを監視し、画像データ（PrtScやコピーされた画像）を検知して自動的に連番JPEGファイルとして保存するGUIツールです。

スクリーンショットの連続保存や、OCR用の画像素材収集（自炊補助）などの作業効率化を目的にPythonで開発しました。

## Features (特徴)

* **自動監視 & 保存**: 「監視開始」ボタンを押すだけでバックグラウンドでクリップボードを監視します。
* **PrtSc連動**: `PrtSc` キーや `Alt + PrtSc`、ブラウザからの画像コピーに反応して即座に保存します。
* **連番管理**: 保存先フォルダ（`Pictures`）内の既存ファイルをチェックし、`0001.jpg` から続く連番でファイル名を生成します。
* **重複回避**: クリップボードに同じデータが残っている間は、重複して保存しません。
* **テキスト無視**: テキストデータのコピーには反応せず、画像のみを保存対象とします。

## Requirement (動作環境)

* Windows 10 / 11
* Python 3.x
